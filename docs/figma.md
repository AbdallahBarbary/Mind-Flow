# Figma Design System

Figma file: https://www.figma.com/design/ZZkjYdpPGecobyC0yhxR14

## Created In Figma

- Product strategy and experience direction.
- Color variables for black, white, neon blue, surfaces, borders, and muted typography.
- Spacing, radius, and typography variables.
- Paint styles, text styles, and elevation styles.
- Auto-layout component variants:
  - Button
  - Input
  - Analytics card
  - Focus timer
  - Modal
  - Navigation bar

## Account Limitation

The Figma Starter plan blocked additional MCP write calls after the design-system pass. Because of that, the full screen pass and prototype implementation were completed directly in the Expo app and documented in code. The Figma file remains the source for foundations and reusable design-system direction.
